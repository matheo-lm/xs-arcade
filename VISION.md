# VISION

## Mission
Build a safe, playful web arcade where kids ages 4-8 can learn through short, high-feedback games.

## Audience
- Primary: children ages 4-8.
- Secondary: parents and educators selecting age-appropriate activities.

## Product Principles
1. Play first: every learning goal is delivered through gameplay, not worksheets.
2. Kid clarity: large touch targets, short labels, strong visual cues.
3. Safe by default: no ads, no third-party analytics, no account requirement.
4. Low friction: instant play in browser on desktop and mobile.
5. Consistent progression: stars and badges across all games.

## Success Outcomes
- Kids can start a game in under 10 seconds.
- Parents can find age-appropriate games through filters quickly.
- Progress persists locally by child profile.
- Each game clearly maps to one or more learning goals.

## Non-Goals (Current Phase)
- Cloud accounts and sync.
- Multiplayer.
- Parent gate flows.
- Full offline-first guarantee.

## 9-Game Learning Portfolio
1. Fruit Stacker: spatial planning and cause/effect.
2. Number Garden: counting and number recognition.
3. Shape Builder: shape recognition and geometry basics.
4. Pattern Parade: sequencing and logic patterns.
5. Memory Trails: working memory and recall order.
6. Letter Lanterns: letter recognition.
7. Phonics Pop: beginning sounds.
8. Word Match: vocabulary and word-picture mapping.
9. Color Craft: creativity and color understanding.

## Roadmap
1. Foundation: launcher, shared modules, one migrated playable game.
2. Content expansion: implement gameplay for remaining eight slots.
3. Progression depth: richer badges and adaptive difficulty tuning.
4. Platform hardening: CI, accessibility audits, performance budgets.
